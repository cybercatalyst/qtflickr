Detailed Description

Class is used to collect data from the parser, it will return only those values that has been setted with the Request object
Example: 
 <?xml version="1.0" encoding="utf-8" ?>
 <rsp stat="ok">
        <photos page="2" pages="89" perpage="10" total="881">
                <photo id="2636" owner="47058503995@N01"/>
                <photo id="2635" owner="47058503995@N01"/>
        </photos>
 </rsp>
If you want to get page number from XML above: 
 QString page = response.attrValue("photos","page");
 qDebug()<<page //output: 2
