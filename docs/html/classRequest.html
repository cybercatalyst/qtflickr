Detailed Description

Use this class to provide information for the XML parser, parser will search only those tags that you have been provided.
Example: 
 <?xml version="1.0" encoding="utf-8" ?>
 <rsp stat="ok">
        <photos page="2" pages="89" perpage="10" total="881">
                <photo id="2636" owner="47058503995@N01"/>
                <photo id="2635" owner="47058503995@N01"/>
        </photos>
 </rsp>
Let assume server response is a XML above and you want to get all id's and owner information for the photo list. The code will look like this: 
 Request request("photo","id,owner");

 QtFlickr qtFlickr("apiKey","apiSecret");
 qtFlickr.get(Method("flickr.photos.getWithGeoData"),request);
