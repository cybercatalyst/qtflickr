Member Data Documentation

QMap<QString, QString> PhotoData::extras

Currently supported fields are: license, date_upload, date_taken, owner_name, icon_server, original_format, last_update, geo, tags, machine_tags, o_dims, views, media, path_alias, url_sq, url_t, url_s, url_m, url_o Note that "geo" field returns "latitude" and "longtitude". Note that "o_dims" field returns "o_width" and "o_height". Usage example: 

 qDebug()<<extras["url_t"];             //prints url of tiny photo
