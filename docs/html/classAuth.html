Detailed Description

flickr.auth methods.

Usage example: 

 UiClass::UiClass()
 {
                ...
                connect(flickr->auth(),SIGNAL(frobFetched (const QString &, FlickrError*)),
                                this,SLOT(frobFetched (const QString &, FlickrError*)));

                flickr->auth()->getFrob();
 }

 void UiClass::frobFetched ( const QString &frob, FlickrError *err )
 {
                if(!err->code){
                        QUrl url = flickr->auth()->getAuthUrl(frob);
                        QDesktopServices::openUrl(url);
                }
 }
All methods calls will return if object status is not "NotRunning", so make sure to check current status of the object by calling getStatus() 
 if(flickr->auth()->getStatus != Auth::NotRunning)
                flickr->auth()->getFrob();
